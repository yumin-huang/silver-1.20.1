A Minecraft Forge mod, by Yumin, that adds Silver Ore and Silver Coins to Minecraft, introducing a lightweight and flexible economy system for player trading and financial transactions.

## 📖 Story & Background

Around October during the first semester of my freshman year in university, my existing Minecraft server reached a point where we needed a dedicated medium for in-game trading and financial transactions.

However, since our server allowed the use of X-ray mods, relying on standard vanilla ores for currency led to heavy inflation. Furthermore, vanilla ores already had practical crafting uses, making them unsuitable as a pure economic tool. To address this issue, I created this mod by following online tutorials and integrating my own design ideas.

Currently, as the server has transitioned into Creative mode, active development for this mod is temporarily paused. If I host a Survival server again in the future, I may continue updating and expanding this project.

## ✨ Features

- ⛏️ **New Ores**: Generates **Silver Ore** and **Deepslate Silver Ore** naturally underground.
- 🔥 **Smelting Support**: Refine raw silver ores into **Silver Ingots** using a standard Furnace or Blast Furnace.
- 🪙 **Currency & Storage Blocks**:
  - Craft **Silver Coins** from Silver Ingots to use as a lightweight currency for player trade.
  - Compact Silver Ingots into **Silver Blocks** for easier resource storage and building.
